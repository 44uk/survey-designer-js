Survey Designer JS

JavaScriptで動作する高度なアンケートをプログラミング不要で作成するためのシステムです。
アンケートの作成はGUIで行います。

特徴

作成可能な設問

以下の設問を簡単に作成することが可能です。各設問には設問タイトル・補足説明を記載することが可能です

  • 基本設問
    • 複数選択肢
    • 単一選択肢
    • 数値記入
    • 1行テキスト
    • 複数行テキスト
    • 表形式
    • 説明文
  • 拡張設問
    • 調査許諾
    • 日程
    • 個人情報
条件分岐

ユーザの入力値に応じて遷移するページを分岐することができます。

SCREENとCOMPLETE

終了ページにSCREENとCOMPLETEページの2種類を設定することができます。ページ分岐を経由して最終的に到達した終了ページによってユーザがSCREEN回答かCOMPLETE回答かを判別できます。

入力値の再掲処理

ユーザが入力した値を後続するページで表示することができます。

ロジック変数

後続の分岐条件や再掲処理で使用することが可能な変数を定義することができます。ユーザの入力値を足したり、引いたりしたものを格納することができます。

さらに高度な利用(JavaScript)

既存の仕組みで対応できなかったことを考えて、ページ内でJavaScriptを設定することができます。JavaScriptは下記のタイミングで実行できます。

  • ページが描画されたとき
  • ページが次のページに行く直前
  • バリデーション時。サーバサイドに問い合わせを行うようなバリデーションも作成可能です。

起動方法

オンラインサンプル

こちらからオンラインサンプルを試すことができます。

制限事項

オンラインサンプルでは下記の機能を実行することができません。
  • 作成したアンケートの保存
  • パネルの選択
  • 配信依頼確認
起動方法

動作確認環境

  • node.js 7.4.0
  • yarn 0.22.0

ローカル環境で動かす

$ git clone https://github.com/jirokun/survey-designer-js.git
$ cd survey-designer-js
$ yarn
$ yarn start

使い方

サーバサイド実装は今のところ存在していません。そのため、Survey Designer Jsに合う形でサーバサイドを実装して組み込む必要があります。

時間ができればサーバサイドの実装も作りたいですが、時間が取れないためいまのところ予定はたっておりません。
